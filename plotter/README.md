# Plotter

`plotter/` is the visualization namespace for lo2cin4bt.

- `plotter/web/` contains the current React + Vite frontend.
- `plotter/web/dist/` is generated by `npm run build` and is ignored by Git.
- Legacy Python Dash/Plotly plotter modules were removed after the React
  frontend became the supported visualization entrypoint.

Run the frontend build from the repo root with:

```bash
cd plotter/web
npm ci
npm run build
```
