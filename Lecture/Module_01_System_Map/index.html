Module 01

系統地圖

Lo2cin4BT 的主線可以先看成一條管線： dataloader -> factorhandler -> backtester -> wfanalyser -> statanalyser -> app API/UI。 每一層只負責自己的工作，讓策略研究、回測記帳、穩健性驗證和報告閱讀可以分開檢查。

模組責任

模組用途注意事項
dataloader/資料來源抽象層。後續模組不會直接呼叫券商 / 交易所 API。
factorhandler/factor 執行基礎。完整 IC、Rank IC、Fama-MacBeth、risk model 屬尚未開發階段。
backtester/統一的向量預計算與逐步投資組合記帳。單資產可作為單資產投資組合。
wfanalyser/WFA 被選中的最優參數與滾動驗證。IS 尋優，OOS 只測被選中的策略。
app/api//plotter/web/FastAPI + React 報告平台。Predictor Analysis 是舊別名；Factor Analysis 因子分析是前端目標命名。

統一回測引擎

strategy_run 表達單資產訊號、calendar/session、多資產投資組合、固定配置與輪動等已支援策略。 不同策略不再各自維護一套互不相干的回測邏輯，而是先把策略轉成同一種目標權重 / 投資組合策略， 再用同一套記帳計算 cash、fees、turnover、holdings、equity 與 benchmark。 特徵、signal、ranking、目標權重可向量化；cash、fees、turnover、holdings、drift、再平衡狀態由逐步記帳處理。
