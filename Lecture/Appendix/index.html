常見問題與術語表

常見問題

我是否需要先懂程式？ 不需要。新手可以先把策略想法交給 AI，讓 AI 按照 repo 內的 agent、skill 和設定檔規範工作。

我應該把自己的策略和資料放在哪裡？ 一般策略研究應放在 workspace/。配置檔放 workspace/runs/，WFA 配置放 workspace/wfa/，本機資料放 workspace/datasets/。

把 CSV 放進 workspace 就會自動被使用嗎？ 不會。資料需要被 workspace/runs/ 設定檔或 workspace/features/ 特徵合約明確引用時才會被調動。

lo2cin4bt 可以下單嗎？ 不可以。lo2cin4bt 目前不是實盤交易或下單系統；券商或交易所帳戶只可作讀取行情資料用途。

回測結果是否代表未來會賺錢？ 不代表。回測只是在指定資料和規則下重放歷史，仍需要檢查資料來源、成本、滑價、過度擬合、look-ahead bias 和 WFA 結果。

術語表

術語意思注意事項
配置檔 (config)描述策略、資料、成本、執行方式和回測範圍的檔案。AI 不應自行發明配置欄位。
Parameter Matrix 參數矩陣一次測試多組參數，例如不同均線長度。最佳參數不是 WFA 證明。
WFA 滾動驗證把歷史切成訓練區間和樣本外區間，觀察策略是否穩定。仍然不是未來收益保證。
Benchmark 基準用來比較策略的參考表現，例如買入並持有 QQQ。基準要和策略的資產、時間、價格口徑相配。
Transaction Cost 交易成本每次買賣扣除的成本假設。成本太低會令結果過度樂觀。
Slippage 滑價成交價和理想價格之間的差距假設。流動性差或高頻策略尤其需要留意。
Look-ahead Bias 偷看未來回測用了當時尚未知道的資料。外部資料必須說明何時可見、何時可交易。
Sharpe 夏普率以波動調整後的報酬指標。不能單獨代表策略可靠。
Max Drawdown 最大回撤資金曲線由高位跌到低位的最大跌幅。是理解風險的重要指標。
