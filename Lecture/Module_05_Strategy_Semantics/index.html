Module 05

策略類型與系統對應

Lo2cin4BT 支援的是能轉換為設定合約、資料來源、回測引擎與輸出檔案的策略。使用時，AI 將會先判斷支援狀態，再產生 config。

目前支援

  • 單資產作為投資組合：MA cross、MA-above、入場 / 出場 signal。
  • 日曆 / 交易時段事件：weekday / ordinal event、同交易時段開倉到收市平倉、自訂事件日期 CSV。
  • 多資產投資組合：固定權重配置、VOO/GLD 輪動、top-N 排名、再平衡。
  • 配置方案、Parameter Matrix 參數矩陣、WFA 滾動驗證。
  • 資料來源：yfinance、Binance、Coinbase、file、FUTU 轉接器框架、IBKR 轉接器框架。

策略描述與系統對應

使用者說法Lo2cin4BT 對應
QQQ 快慢均線交叉，掃描 10 到 200 天。single_asset_signal + parameter_matrix + 單資產投資組合。
每個月第一個星期一開市交易 QQQ，收市平倉。calendar_event_session + 交易所交易日曆 + 開盤 / 收盤執行。
VOO/GLD momentum 排名，只持 top 1。multi_asset_portfolio + ranking + eligibility + allocation。
固定權重年度再平衡。固定權重配置；無參數範圍時使用 Rolling Validation 滾動驗證。

不支援策略的處理

如果提出完整實盤下單路由、tax lot/corporate action、完整 factor IC/Fama-MacBeth/risk model、未實作券商成交模型，AI 將會說明不支援、需要澄清或需要開發。
