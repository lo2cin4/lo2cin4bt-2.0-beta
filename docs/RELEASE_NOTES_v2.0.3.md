# lo2cin4bt 2.0.3 Release Notes

Beta patch release for public testing.

## Fixed

- Fixed the Strategy Performance run picker so Walk-Forward Analysis results stay in the WFA page instead of appearing as normal strategy-performance runs.
- Fixed the optional Windows desktop shortcut launcher so it clears stale local lo2cin4bt backend processes before starting the app and shows startup errors in the command window.
- Extended Mosaic screenshot mode to hide strategy names, selected strategy logic, parameter-candidate names, top-ranked parameter summaries, and parameter importance.

## Safety Notice

lo2cin4bt is for local research, teaching, and backtesting only. It is not
investment advice, not live-trading software, and not an order-routing system.
It must not be used to place orders, move funds, change positions, or change
broker or exchange account settings.
