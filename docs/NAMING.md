# lo2cin4bt Naming Notes

## Universe Constituents

Use this user-facing term:

- Historical Universe Constituents
- 歷史成份清單

Avoid old user/member-list wording. In this project the concept is the
historical constituent set of a tradable universe, such as which symbols
belonged to an index, sector, curated universe, or tradable list on each date.

Preferred public/config names for future contracts:

- `historical_constituents_path`
- `historical_constituents_hash`
- `universe_constituents_path`
- `universe_constituents_hash`

Existing internal compatibility aliases may remain only for migration reads, but
new docs, schemas, and user-facing copy should say Historical Universe
Constituents / 歷史成份清單.
