# Agent Reports / AI 代理報告

This folder stores durable AI-agent reports that belong to this repo-level workspace.

此資料夾用來保存屬於本 repo 工作區的 AI 代理報告。

## Recommended Path / 建議路徑

Use one folder per agent:

每個代理使用一個資料夾：

- `reports/agents/<agent_name>/YYYY-MM-DD_<short-topic>.md`

Example / 例子：

- `reports/agents/lo2cin4bt_PM/2026-06-05_workspace-folder-map.md`
- `reports/agents/lo2cin4bt_StrategyBuilderSubAgent/2026-06-05_ipo-breakout-review.md`

## What Belongs Here / 適合放在這裡

- repo-local review notes
- strategy-building notes
- acceptance findings
- user-facing explanations connected to this workspace

- repo 本地檢查筆記
- 策略建立筆記
- 驗收結果
- 與此工作區相關、可供用戶閱讀的說明

## What Does Not Belong Here / 不適合放在這裡

- runtime outputs
- private datasets
- API keys, credentials, or account screenshots
- generated chart payloads or raw backtest result folders

- 執行輸出
- 私人資料集
- API key、登入憑證或帳戶截圖
- 生成的圖表 payload 或原始回測結果資料夾
