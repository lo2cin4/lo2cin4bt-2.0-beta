# Reports / 本地報告

`reports/` stores local notes, AI review outputs, and repo-level research reports.

`reports/` 用來放本地筆記、AI 審查輸出和 repo 內研究報告。

## Put Here / 可以放這裡

- agent review notes
- strategy explanation drafts
- local experiment summaries
- checklist results

- agent 審查筆記
- 策略解釋草稿
- 本地實驗摘要
- checklist 結果

## Do Not Put Here / 不應放這裡

- private keys
- broker account details
- raw large datasets
- generated app outputs

- 私鑰
- 券商帳戶資料
- 大型原始數據
- app 自動生成輸出

## Structure / 結構

Use `reports/agents/` when the report was written by an AI agent for this repo.

如果報告是 AI agent 為這個 repo 寫的，請放在 `reports/agents/`。
